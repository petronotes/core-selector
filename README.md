# Core Selector 🪨

**Инструмент для выделения областей на фотографиях керна с экспортом в CSV.**

🔗 **[Открыть онлайн](https://petronotes.github.io/core-selector/)**

---

## Возможности

- 📷 Загрузка фото керна (drag & drop или кнопка)
- 🪄 Инструмент «Волшебная палочка» — выделение по цвету
- 🎨 Несколько независимых областей выделения с подсветкой
- ✏️ Редактирование областей: добавить / вычесть пиксели
- 📏 Ввод кровли и подошвы в метрах для масштабирования
- 📊 Экспорт результата в CSV (глубина + бинарный признак через 1 см)
- 🌙 Тёмная / светлая тема

## Алгоритм экспорта CSV

1. Для каждой строки пикселей определяется доля выделенных пикселей
2. Высота фото разбивается на окна по 1 см: `px/см = высота_фото / ((подошва - кровля) × 100)`
3. В каждом 1-сантиметровом окне: среднее по строкам ≥ 0.5 → `1`, иначе → `0`
4. Итоговый CSV: `depth_m` (глубина от кровли через 1 см) + `selected` (0 или 1)

### Пример вывода

```csv
depth_m,selected
1200.00,0
1200.01,1
1200.02,1
1200.03,0
...
```

## Использование

### Онлайн (GitHub Pages)

Просто откройте ссылку выше — никакой установки не нужно. Всё работает в браузере.

### Локально

```bash
git clone https://github.com/YOUR_USERNAME/core-selector.git
cd core-selector
# Открыть index.html в браузере — никаких зависимостей не нужно
open index.html  # macOS
start index.html # Windows
```

## Развёртывание на GitHub Pages

1. Создайте репозиторий на GitHub (можно приватный с Pages на публичной ветке)
2. Загрузите файлы:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/core-selector.git
   git push -u origin main
   ```
3. Перейдите в **Settings → Pages**
4. Выберите **Source: GitHub Actions** (или **Deploy from branch: main / root**)
5. Через ~1 минуту сайт будет доступен по адресу `https://YOUR_USERNAME.github.io/core-selector/`
